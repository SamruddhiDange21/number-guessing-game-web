# Number Guessing Game (Web)

## Description
This project is a simple web-based number guessing game built using HTML, CSS, and JavaScript.

The game generates a random number between 1 and 100 and the player must guess the correct number. The program provides feedback if the guess is too high or too low and keeps track of the number of attempts.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Random number generation
- Hint system (Too High / Too Low)
- Attempt counter
- Play Again button to restart the game

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Start guessing the number.

## Example

Guess: 50  
Output: Too Low  

Guess: 75  
Output: Too High  

Guess: 63  
Output: Correct!
